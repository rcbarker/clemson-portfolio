Slot: primary_control

Documentation: Slot that uses seven values to show the primary control knob of an effects unit. Symbol: - Rate: The pedal's primary control manipulates the rate of an effect in the signal chain. - Gain: The pedal's primary control manipulates a gain value in the signal chain. - Pitch: The pedal's primary control manipulates the pitch an effect in the signal chain. - Depth: The pedal's primary control manipulates the depth of an effect in the signal chain. - Echo: The pedal's primary control manipulates the timing of an echo signal in the signal chain. - Speed: The pedal's primary control manipulates the speed of an effect in the signal chain. Different from rate; see Phaser pedal documentation. - Range: The pedal's primary control manipulates the frequency range of an effect in the signal chain.
